# Redfin facing filter (modular)

This is a refactor of `redfin_facing_2.html` into **HTML + CSS + ES modules**.

## Run

Because browsers often block ES modules from `file://`, run a local server:

```bash
cd "/Users/venkat/Cursor Projects/redfin_facing_modular"
python3 -m http.server 5173
```

Then open `http://localhost:5173`.
